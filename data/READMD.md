# N-CMAPSS Dataset

[N-CMAPSS](https://data.nasa.gov/dataset/cmapss-jet-engine-simulated-data) is a multivariate time-series engine dataset.

The data contains training and test sets, each row represents cycle information with 26 columns of numbers.

The columns correspond to: 1) unit number 2) time, in cycles 3) operational setting 1 4) operational setting 2 5) operational setting 3 6) sensor measurement 1 7) sensor measurement 2 ... 26) sensor measurement 26.

- [Download Link](https://data.nasa.gov/docs/legacy/CMAPSSData.zip)

For more information, refer to `CMAPSSData/readme.txt` file from the download.
